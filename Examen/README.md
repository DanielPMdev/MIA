# Exámenes

> Asignatura: Modelos de Inteligencia Artificial  
> Curso de Especialización en Inteligencia Artificial y Big Data

## 📌 Descripción
Esta carpeta contiene el material de preparación y estudio para los exámenes de la asignatura, organizados por convocatoria. Incluye apuntes recopilatorios, ejercicios resueltos y preguntas tipo test que abarcan los contenidos de las diferentes unidades didácticas.

## 📁 Estructura

### Examen Enero
Material de estudio para la convocatoria de enero, que cubre las unidades UD1 a UD5:
- **Apuntes**: Resúmenes de las unidades UD1 (Introducción a la IA), UD2 (Tipos de IA), UD3 (Sistemas Expertos), UD4 (Sistemas Cognitivos) y UD5 (Principios Legales), incluyendo un documento específico sobre anonimización.
- **Ejercicios**: Actividades resueltas de las unidades 1, 2 y 4.
- **Test**: Preguntas tipo test y material de los PDFs originales de cada unidad.
